# petruciOS

A personal web-based operating system built with HTML, CSS, and JavaScript. This is a showcase of my personality, interests, and projects, all wrapped in a retro OS-like interface.

## Features

- **Desktop Environment** - full-screen desktop with wallpaper, icons, and taskbar
- **Draggable Windows** - all windows can be moved by dragging the title bar
- **Welcome Screen** - personal intro with links to GitHub and Instagram
- **Notes App** - write, save, and delete notes (persistent during the session)
- **Tic-Tac-Toe** - play against a friend on a 3x3 board with win detection
- **Settings** - change the desktop background theme (Dark Blue, Dark Gray, Forest, Crimson)
- **Live Clock** - real-time clock displayed in the taskbar
- **Bring to Front** - click any window to bring it to the foreground

## How to Use

Open `index.html` in any browser. Click desktop icons or taskbar items to open windows. Drag windows by their title bars. Close windows with the X button.

or just go to https://webos-eight-eta.vercel.app/


## Tech Stack

- HTML5
- CSS3 (inline styles, flexbox, gradients)
- Vanilla JavaScript (DOM manipulation, event handling)
